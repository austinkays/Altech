Internal Tools Acknowledgment

Altech Insurance Agency

Tool: Altech Toolkit (altech.agency)
Purpose: An internal insurance-agency workflow tool used by Altech Insurance Agency staff to capture client intake information, track CGL / General Liability renewal compliance, generate carrier-ready export files (EZLynx XML, HawkSoft CMSMTF, PDF), and assist with day-to-day client communication. Client data is encrypted with AES-256-GCM before it is synced to the cloud (Supabase). The full technical security model is documented on the in-app Security & Data Protection page (padlock icon in the sidebar footer).
