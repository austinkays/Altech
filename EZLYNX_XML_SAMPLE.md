# EZLynx XML Export Sample

This document provides a sample of the XML export format generated by the Altech Field Lead application for EZLynx integration.

## XML Format

The XML export follows the EZLynx EventCallbackNotification format for applicant data import:

```xml
<?xml version="1.0"?>
<EventCallbackNotification xmlns:xsd="http://www.w3.org/2001/XMLSchema" xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance">
    <Name>ApplicantCreated</Name>
    <Entity>Applicant</Entity>
    <EntityId>Smith_1770098980750</EntityId>
    <EventData>
        <DictionaryEntry>
            <Key xsi:type="xsd:string">FirstName</Key>
            <Value xsi:type="xsd:string">John</Value>
        </DictionaryEntry>
        <DictionaryEntry>
            <Key xsi:type="xsd:string">LastName</Key>
            <Value xsi:type="xsd:string">Smith</Value>
        </DictionaryEntry>
        <DictionaryEntry>
            <Key xsi:type="xsd:string">Email</Key>
            <Value xsi:type="xsd:string">john@example.com</Value>
        </DictionaryEntry>
        <!-- Additional fields... -->
    </EventData>
</EventCallbackNotification>
```

## Included Data Fields

The XML export includes all collected data from the application:

### Personal Information
- FirstName, LastName
- Email, Phone
- DateOfBirth
- MaritalStatus, Education, Industry

### Address
- AddressLine1
- City, StateCode, ZipCode

### Property Details
- PropertyPurchaseDate
- DwellingType, DwellingUsage, OccupancyType
- YearBuilt, SquareFootage
- NumStories, FullBaths
- ConstructionStyle, ExteriorWalls, Foundation
- KitchenQuality

### Roof and Systems
- RoofType, RoofShape, RoofYear
- HeatingType, HeatingYear
- CoolingType
- PlumbingYear, ElectricalYear

### Safety and Security
- FireStationDistance, FireHydrantDistance
- ProtectionClass
- BurglarAlarm, FireAlarm, Sprinklers

### Risk Factors
- Pool, Trampoline, WoodStove
- DogInfo
- BusinessOnProperty

### Auto and Driver Information
- DriverLicenseNumber, DriverLicenseState
- VIN, VehicleDescription
- VehicleUse
- AnnualMiles, CommuteDistance
- RideSharing, Telematics

### Insurance History
- LiabilityLimits
- HomeDeductible, AutoDeductible
- PriorCarrier, PriorYears, PriorExpiration
- Accidents, Violations
- StudentGPA

### Additional Information
- Mortgagee
- AdditionalInsureds
- PreferredContactTime
- ReferralSource
- TCPAConsent
- QuoteType
- Notes (includes all formatted information)

## Features

- **XML Escaping**: All special characters (&, <, >, ", ') are properly escaped
- **Empty Field Handling**: Fields with no value are not included in the export
- **Notes Field**: Includes all collected information in a formatted text block
- **Unique EntityId**: Generated using last name and timestamp
- **Valid XML**: Conforms to XML 1.0 specification with proper namespaces
