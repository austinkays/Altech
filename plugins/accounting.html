ℹ️ How It Works

This tool uses Playwright (a browser automation library) to open HawkSoft's trust accounting page. You log in manually to handle 2FA — the script then takes over, selecting only Receipt rows and clicking Export → Download. The CSV is saved to your Altech project folder.

⚠️ Desktop only — requires the local server running and Python + Playwright installed.
